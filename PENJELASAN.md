Program ini digunakan untuk menghitung total nilai dan rata-rata nilai sejumlah siswa berdasarkan input nilai yang dimasukkan oleh pengguna

Deskripsi Program :
Pengguna cukup memasukkan jumlah siswa, kemudian menginput nilai tiap siswa satu per satu. Hasil total dan rata-rata akan otomatis dihitung dan ditampilkan

Cara Kerja Program : 
1.  Pengguna diminta memasukkan jumlah siswa
2.  Program melakukan perulangan `for` sebanyak jumlah siswa tersebut
3.  Di setiap perulangan, pengguna diminta menginput nilai siswa
4.  Semua nilai dijumlahkan ke dalam variabel `totalNilai`
5.  Setelah semua nilai dimasukkan, program menghitung rata-rata
